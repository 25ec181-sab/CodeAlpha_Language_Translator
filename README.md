# Language Translation Tool

This project is developed as part of the **CodeAlpha Artificial Intelligence Internship**.

## Description
The Language Translation Tool allows users to translate text from one language to another using AI-based NLP translation APIs.

## Technologies Used
- Python
- Streamlit
- Deep Translator (Google Translator API)

## Features
- User-friendly interface
- Supports multiple languages
- Real-time translation

## How It Works
1. User enters text
2. Selects source and target language
3. AI translates and displays the output

## How to Run
pip install -r requirements.txt  
streamlit run app.py
